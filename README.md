## 📌 Tabuada em Python

Este projeto consiste em uma **tabuada simples em Python**, onde o usuário informa um número e o sistema exibe a multiplicação desse número de **1 até 10**.

Além disso, o projeto conta com a integração de um **fluxo de Integração Contínua (CI)** utilizando **GitHub Actions**, garantindo que o código seja automaticamente testado a cada alteração.

---

## 🚀 Funcionalidades

* Entrada de um número pelo usuário
* Geração da tabuada de 1 a 10
* Exibição formatada dos resultados
* Testes automatizados com `pytest`
* Execução automática dos testes via GitHub Actions

---

## ⚙️ Tecnologias utilizadas

* Python 3
* Pytest
* GitHub Actions

---

## 🧪 Integração Contínua

O projeto utiliza **CI (Continuous Integration)** para:

* Executar testes automaticamente
* Garantir que o código esteja funcionando corretamente
* Evitar erros em novas alterações
